Desa Digital Platform

A full-stack web application simulating a digital platform for village services, with a focus on API security. This project showcases skills in both front-end (Vue.js) and back-end (Laravel) development, as well as secure API design.

Project Outline

The Desa Digital platform provides a centralized system for managing village services, such as citizen data, announcements, and local business listings. The architecture emphasizes a decoupled front-end and back-end, communicating through a secure RESTful API.
