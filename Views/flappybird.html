Flappy Bird Clone

A classic game clone built with Python to explore game loops, physics, and collision detection. This project is a fun exercise in understanding the fundamentals of game development and object-oriented programming.

Project Outline

This Flappy Bird clone replicates the core mechanics of the original game. The player controls a bird and must navigate through a series of pipes. The game is built using the Pygame library, providing a hands-on introduction to game development concepts.
